# 💎 Fitlytics AI Pro Max

An advanced AI-powered fitness analytics dashboard that predicts calories burned and visualizes workout performance in real time.

---

## 🚀 Features
- 3D Interactive Workout Visualization
- AI-based Calorie Prediction
- Dark Neon UI Dashboard
- Sidebar Filters (Workout Type, Gender)
- Real-time Data Insights

---

## 🧠 Tech Stack
- Python  
- Pandas  
- Scikit-Learn  
- Streamlit  
- Plotly  

---

## 📊 Project Preview
👉 Run locally:
streamlit run app.py

---

## 💡 Goal
To combine Machine Learning + Data Visualization into a real-world fitness intelligence system.
